Detailed Description

The namespace contains classes that do write/read to/from JSON via either the Json classes in "json.h" or via the io function of the object to be written/read.

Which of those classes is actually used determined via the template parameter and use of SFINAE. Usage: 

io::detail::has_archive_tag<SomeType>::write( stream, instance_of_SomeType );
