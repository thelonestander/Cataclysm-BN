Enumeration (de)serialization to/from string.

enum_to_string converts an enumeration value to a string (which can be written to JSON).

The result must be an non-empty string.

This function must be implemented somewhere for each enumeration type for which conversion is required.

Such enums must also specialize enum_traits to specify their 'last' member.

string_to_enum converts the string value back into an enumeration value. The input is expected to be one of the outputs of enum_to_string. If the given string does not match an enumeration, an InvalidEnumString is thrown.

string_to_enum<E>(enum_to_string<E>(X)) == X

must yield true for all values of the enumeration E.
