38/**
39 * Get a trait group ID and optionally load an inlined trait group.
40 *
41 * If the value is string, it's assumed to be a trait group id and it's
42 * returned directly.
43 *
44 * If the value is a JSON object, it is loaded as a trait group. The group will be given a
45 * unique id (if the JSON object contains an id, it is ignored) and that id will be returned.
46 * If the JSON object does not contain a subtype, the given default is used.
47 *
48 * If the value is a JSON array, it is loaded as a trait group: the default_subtype will be
49 * used as subtype of the new trait group and the array is loaded like the "entries" array of
50 * a trait group definition (see format of trait groups).
51 *
52 * @param default_subtype If an inlined trait group is loaded this is used as the default
53 * subtype. It must be either "distribution" or "collection". See @ref Trait_group.
54 * @throw JsonError as usual for JSON errors, including invalid input values.
55 */
56Trait_group_tag load_trait_group( const JsonValue &value, const std::string &default_subtype );
