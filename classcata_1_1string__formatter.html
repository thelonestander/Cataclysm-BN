Detailed Description

Type-safe and undefined-behavior free wrapper over sprintf.

See string_format for usage. Basically it extracts the format specifiers and calls sprintf for each one separately and with proper conversion of the input type. For example printf("%f", 7) would yield undefined behavior as "%f" requires a double as argument. This class detects the format specifier and converts the input to double before calling sprintf. Similar for printf("%d", "foo") (yields UB again), but this class will just throw an exception.
