5/**
6A pair of classes to provide safe references to objects.
7
8A safe_reference_anchor is made a member of some object, and a safe_reference
9to that object (or any other object with the same lifetime) can be obtained
10from the safe_reference_anchor.
11
12When the safe_reference is destroyed or assigned-to, all safe_references
13derived from it are invalidated. When one attempts to fetch the referenced
14object from an invalidated safe_reference, it returns nullptr.
15
16The motivating use case is to store references to items in item_locations in a
17way that is safe if that item is moved or destroyed.
18
19*/
