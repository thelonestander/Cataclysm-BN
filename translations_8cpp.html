Translation with a gendered context.

Similar to pgettext, but the context is a collection of genders.

Parameters
gendersA map where each key is a subject name (a string which should make sense to the translator in the context of the line to be translated) and the corresponding value is a list of potential genders for that subject. The first gender from the list of genders for the current language will be chosen for each subject (or the language default if there are no genders in common).
