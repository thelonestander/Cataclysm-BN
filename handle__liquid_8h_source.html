33/**
34 * Consume / handle all of the liquid. The function can be used when the liquid needs
35 * to be handled and can not be put back to where it came from (e.g. when it's a newly
36 * created item from crafting).
37 * The player is forced to handle all of it, which may required them to pour it onto
38 * the ground (if they don't have enough container space available) and essentially
39 * loose the item.
40 * @return Whether any of the liquid has been consumed. `false` indicates the player has
41 * declined all options to handle the liquid (essentially canceled the action) and no
42 * charges of the liquid have been transferred.
43 * `true` indicates some charges have been transferred (but not necessarily all of them).
44 */

47/**

51 * It returns when all liquid has been handled or if the player has explicitly canceled
52 * the action (use the charges count to distinguish).
53 * @return Whether any of the liquid has been consumed. `false` indicates the player has
