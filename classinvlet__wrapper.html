Detailed Description

Wrapper to handled a set of valid "inventory" letters.

"inventory" can be any set of objects that the player can access via a single character (e.g. bionics). The class is (currently) derived from std::string for compatibility and because it's simpler. But it may be changed to derive from std::set<int> or similar to get the full range of possible characters.
