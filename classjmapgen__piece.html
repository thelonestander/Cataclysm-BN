Detailed Description

Basic mapgen object.

It is supposed to place or do something on a specific square on the map. Inherit from this class and implement the apply function.

Instructions for adding more mapgen things:

  1. create a new class inheriting from jmapgen_piece.
    • It should have a constructor that a accepts a JsonObject and initializes all it local members from it. On errors / invalid data, call JsonObject::throw_error or similar.
    • Implement the apply function to do the actual change on the map.
  2. Go into mapgen_function_json::setup and look for the lines with load_objects, add a new line with your own class there. It should look like
    load_objects<your_own_class_from_step_1>( jo, "place_something" );
    Use a descriptive name for "something", preferably matching the name of your class.
  3. Go into mapgen_function_json::setup and look for the lines with load_place_mapings, add a new line with your class there. It should look like
    load_place_mapings<your_own_class_from_step_1>( jo, "something", format_placings );
    Using the same "something" as in step 2 is preferred.

For actual examples look at the commits that introduced the load_objects/load_place_mapings lines (ignore the changes to the json files).
