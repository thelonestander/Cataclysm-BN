484// Helper function to convert coordinate delta to a movement action
485/**
486 * Translate coordinate delta into movement action
487 *
488 * For a given coordinate delta, this function returns the associated user movement action
489 * that would generated that delta. See @ref action_id for the list of available movement
490 * commands that may be generated. This function takes iso mode into account.
491 *
492 * The only valid values for the coordinates of \p d are -1, 0 and 1
493 *
494 * @note: This function does not sanitize its inputs, which can result in some strange behavior:
495 * 1. If d.z is valid and non-zero, then d.x and d.y are ignored.
496 * 2. If d.z is invalid, it is treated as if it were zero.
497 * 3. If d.z is 0 or invalid, then any invalid d.x or d.y results in @ref ACTION_MOVE_FORTH_LEFT
498 * 4. If d.z is 0 or invalid, then a d.x == d.y == 0 results in @ref ACTION_MOVE_FORTH_LEFT
499 *
500 * @param[in] d coordinate delta, each coordinate should be -1, 0, or 1
501 * @returns ID of corresponding move action (usually... see note above)
502 */
503action_id get_movement_action_from_delta( const tripoint &d, iso_rotate rot );
