22/**
23 * Reference to a position (a point) of the @ref vehicle.
24 * It does not refer to a specific vehicle part, but to a mount point of a
25 * vehicle that contains one or more vehicle parts.
26 *
27 * It's supposed to be the basic vehicle interface for the @ref map.
28 * Another class shall be used to get a reference to a specific part of the
29 * vehicle.
30 *
31 * Note that it must be created with a valid vehicle reference and a valid
32 * part index. An instance can become invalid when the referenced vehicle is
33 * changed (parts added / removed or whole vehicle removed). There is no way
34 * to detect this (it behaves like C++ references).
