Detailed Description

template<typename range_type>
class vehicle_part_iterator< range_type >

Exposes (multiple) parts of one vehicle as vpart_reference.

Uses an arbitrary range_type to supply

  • the vehicle reference,
  • information which parts should be iterated over (and which should be skipped).
