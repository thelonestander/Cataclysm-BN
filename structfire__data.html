Detailed Description

Contains the state of a fire in one tile on one turn.

Used to pass data between the fire field calculations in map::process_fields_in_submap() and the item::burn() function of each specific item being burned.

A structure of this type is created during the burn calculation for a given turn and a given tile. The calculation iterates through all items on the tile, with this structure serving to accumulate the sum of smoke produced by and fuel contributed by each item in the fire this turn.
